Module H5Z¶

Filter API and constants.

h5py.h5z.filter_avail(INT filter_code) → BOOL¶

Determine if the given filter is available to the library. The filter code should be one of:

  • FILTER_DEFLATE

  • FILTER_SHUFFLE

  • FILTER_FLETCHER32

  • FILTER_SZIP

h5py.h5z.get_filter_info(INT filter_code) → INT filter_flags¶

Retrieve a bitfield with information about the given filter. The filter code should be one of:

  • FILTER_DEFLATE

  • FILTER_SHUFFLE

  • FILTER_FLETCHER32

  • FILTER_SZIP

Valid bitmasks for use with the returned bitfield are:

  • FILTER_CONFIG_ENCODE_ENABLED

  • FILTER_CONFIG_DECODE_ENABLED

h5py.h5z.register_filter(INT cls_pointer_address) → BOOL¶

Register a new filter from the memory address of a buffer containing a H5Z_class1_t or H5Z_class2_t data structure describing the filter.

cls_pointer_address can be retrieved from a HDF5 filter plugin dynamic library:

import ctypes

filter_clib = ctypes.CDLL("/path/to/my_hdf5_filter_plugin.so")
filter_clib.H5PLget_plugin_info.restype = ctypes.c_void_p

h5py.h5z.register_filter(filter_clib.H5PLget_plugin_info())
